
# git bash 사용법

git client 도구들만 사용하다보니, git bash 는 맨날 까먹는다.
자주 쓰이는데 자꾸 까먹는 git bash 사용법을 정리해 봐야겠다. 

## git clone

1. 로컬에 clone 받고싶은 폴더를 만든다. (영어로)
2. git bash 에 들어간다. 
3. 거기서 해당 폴더까지 들어간다.
    <img src="../data/image/gi1.PNG">

4. 거기서 `git clone` 하고 git repository 의 ssh 주소를 복사한다. 
    중간에 나오는거에 yes 해주기

그러면 clone 완료!!
